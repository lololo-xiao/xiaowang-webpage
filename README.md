# xiaowang.lol

Minimal personal site. Black-and-white, e-ink aesthetic.

## Run locally

```bash
npm install
npm run dev
```

## Structure

- `app/page.tsx` — single page with about & contact sections
- `app/globals.css` — e-ink styling, grain texture, animations
- `app/layout.tsx` — root layout with metadata

## Deploy

Push to GitHub, import into Vercel. Add domain in Vercel settings.
